About This API Playground

This playground offers a way to quickly test the Notes App API. While the real API requires authentication, this environment is open and does not require any login.

Heads up! This is a public playground. Any notes you create or media you upload will be visible to other users, and you will see their data as well. Please do not use any sensitive or personal information here.
